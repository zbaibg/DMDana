DMDana.lib.matana.DMD_MatrixAnalyzer

class DMD_MatrixAnalyzer(DMDfolder)

Bases: object

Class for analyzing DMD (Density Matrix Decomposition) matrix data from electronic structure computations.

Parameters:

DMDfolder (str) – Path to the folder containing the DMD data files.

Variables:

  • kmesh – Number of k-points in each direction (x, y, z).

  • nb – Number of bands.

  • bBot_dm – Lower band index for DMD.

  • bTop_dm – Upper band index for DMD.

  • nb_dm – Number of DMD bands.

  • nk – Number of k-points.

  • kstep – Step size between k-points in each direction.

  • denmat – Density matrix from DMD calculations.

  • vmat – Potential matrix from DMD calculations.

  • Emat – Energy matrix from DMD calculations.

  • kveclist – List of k-vectors.
