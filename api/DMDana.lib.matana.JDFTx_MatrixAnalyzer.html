DMDana.lib.matana.JDFTx_MatrixAnalyzer

class JDFTx_MatrixAnalyzer(folder, kmesh_num, nb_dft)

Bases: object

Class for analyzing JDFTx matrix data related to electronic structure calculations.

Parameters:

  • folder (str) – The folder path containing the data files.

  • kmesh_num (Tuple[int, int, int]) – The number of k-points in each direction (knx, kny, knz).

  • nb_dft (int) – The number of DFT bands.

Variables:

  • vmat_dft – Matrix of DFT data loaded from files.

  • kveclist – List of k-vectors associated with the matrix.

  • vmat_minus_k – Matrix of -k corresponding to the given k-points.
