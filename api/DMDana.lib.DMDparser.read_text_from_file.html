DMDana.lib.DMDparser.read_text_from_file

read_text_from_file(filepath, marklist, locationlist, stop_at_first_find, dtypelist=<class 'str'>, sep=None)[source]

Read text from a file based on specified markers and locations.

Parameters:

  • filepath (str) – The path to the file to read from.

  • marklist (List[str]) – A list of markers to search for in the file.

  • locationlist (List[int]) – A list of locations corresponding to the markers.

  • stop_at_first_find (bool) – Whether to stop searching after the first match for each marker.

  • dtypelist (type) – The data type or a list of data types to convert the results to (default: str).

  • sep (str) – The separator to use when splitting lines (default: None).

Returns:

A list of results extracted from the file.

Raises:

AssertionError – If marklist and locationlist have different lengths, or if dtypelist is not a type or list of types.

Return type:

List

Usage: To get mu and temperature from ldbd_size.dat, in which the content is like: 9.50043414701698e-04 # T 0.00000000000000e+00 0.00000000000000e+00 0.00000000000000e+00 # muMin, muMax mu just use: mu, temperature = read_text_from_file(‘ldbd_data/ldbd_size.dat’, [“mu”, “# T”], [2, 0]) Remember to manually convert the string to the data type you want. dtypelist could either be a type or a list of types.
